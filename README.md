# 英会話教室 レベルチェック機能 - サンプル版

## 📋 概要

会社への提案用デモンストレーションとして作成した、ブラウザ上で動作するサンプルバージョンです。

## 🚀 使い方

1. `index.html` をブラウザで開く
2. 「診断を始める」ボタンをクリック
3. 用途を選択（複数選択可）
4. 5問の診断問題に回答
5. 診断結果を確認
6. 問い合わせフォームに入力（デモ用）

## 📁 ファイル構成

```
レベル判定/
├── index.html          # メインHTMLファイル
├── styles.css          # スタイルシート
├── script.js           # JavaScript機能
├── README.md           # このファイル
├── サンプル版_要件定義書.md
└── 提案書.md           # 提案書
```

## ✨ 機能

- ✅ ランディング画面
- ✅ 用途選択（旅行、ビジネス、日常会話、その他）
- ✅ レベル診断（5問、用途に応じた問題）
- ✅ 診断結果表示（レベル、おすすめコース、診断結果ID）
- ✅ 問い合わせフォーム（バリデーション付き）
- ✅ 送信完了画面

- **明るく楽しい雰囲気**: グラデーション背景、アニメーション

- HTML5
- CSS3（アニメーション、グラデーション）
- JavaScript (Vanilla JS)

## 🌐 デプロイ方法

### Netlify
1. [Netlify](https://www.netlify.com/) にアカウント作成
2. このフォルダをドラッグ&ドロップ
3. 自動的にデプロイ完了

### Vercel
1. [Vercel](https://vercel.com/) にアカウント作成
2. GitHubにリポジトリを作成してプッシュ
3. Vercelでインポート

### GitHub Pages
1. GitHubにリポジトリを作成
2. Settings > Pages で公開

## 📝 注意事項

- **デモ版**: 実際のメール送信機能は実装されていません
- **データ保存**: ローカルストレージのみ（ブラウザに保存）
- **診断結果ID**: ランダム生成（デモ用）

## 🔄 本番版との違い

| 項目 | サンプル版 | 本番版 |
|------|-----------|--------|
| メール送信 | なし（デモのみ） | あり（実際に送信） |
| データベース | なし | あり（サーバーに保存） |
| 診断結果ID | ランダム生成 | 一意のID生成 |
| 問題数 | 5問 | 5-7問 |
| 用途選択 | 4個 | 6個 |
