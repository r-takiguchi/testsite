# デモサイトを会社で見てもらう方法

## ⚠️ セキュリティの高い会社のPCの場合

**推奨方法：USBメモリまたはローカルネットワーク共有**

このサイトは**完全にオフラインで動作**します。外部サーバーへの接続は一切不要です。

### ✅ セキュリティの高い会社でも確実に動作する方法

#### 方法A: USBメモリで共有（最も確実）

1. **USBメモリにファイルをコピー**
   - 以下の3つのファイルをUSBメモリにコピー：
     - `index.html`
     - `styles.css`
     - `script.js`

2. **各PCで開く**
   - USBメモリを挿入
   - `index.html` をダブルクリック
   - ブラウザで自動的に開きます

**メリット：**
- ✅ インターネット接続不要
- ✅ 外部サービス不要
- ✅ セキュリティポリシーに影響しない
- ✅ 最も確実に動作する

**注意点：**
- 会社のセキュリティポリシーでUSBメモリの使用が禁止されている場合は使用不可

---

#### 方法B: ローカルネットワーク共有（会社内のみ）

1. **共有フォルダに配置**
   - 会社の共有フォルダ（例: `\\server\shared\`）に以下のファイルをコピー：
     - `index.html`
     - `styles.css`
     - `script.js`

2. **共有パスを伝える**
   - 例: `\\server\shared\english-level-check\`
   - このパスを会社の人に共有

3. **各PCで開く**
   - エクスプローラーで共有フォルダを開く
   - `index.html` をダブルクリック

**メリット：**
- ✅ インターネット接続不要
- ✅ 会社内のみで安全
- ✅ USBメモリ不要

**注意点：**
- 会社のネットワーク設定が必要
- 外部からはアクセス不可

---

## 方法1: GitHub Pages（インターネット接続が必要）

### 手順

1. **GitHubアカウントを作成**（まだの場合）
   - https://github.com にアクセス
   - 無料アカウントを作成

2. **新しいリポジトリを作成**
   - GitHubにログイン
   - 右上の「+」→「New repository」
   - Repository name: `english-level-check`（任意の名前）
   - Public を選択
   - 「Create repository」をクリック

3. **ファイルをアップロード**
   - 作成したリポジトリのページで「uploading an existing file」をクリック
   - 以下のファイルをドラッグ&ドロップ：
     - index.html
     - styles.css
     - script.js
   - 「Commit changes」をクリック

4. **GitHub Pagesを有効化**
   - リポジトリの「Settings」タブをクリック
   - 左メニューの「Pages」をクリック
   - Source: 「main」ブランチを選択
   - 「Save」をクリック

5. **URLを確認**
   - 数分待つと、以下のようなURLが表示されます：
     `https://[あなたのユーザー名].github.io/english-level-check/`
   - このURLを会社の人に共有するだけ！

### メリット
- ✅ 無料
- ✅ URLを共有するだけで誰でも見られる
- ✅ スマホからも見られる
- ✅ 更新も簡単（ファイルをアップロードし直すだけ）

## 方法2: Netlify Drop（超簡単・GitHub不要）

### 手順

1. **Netlify Dropにアクセス**
   - https://app.netlify.com/drop にアクセス

2. **ファイルをドラッグ&ドロップ**
   - 以下の3つのファイルをまとめてドラッグ&ドロップ：
     - index.html
     - styles.css
     - script.js
   - 自動的にデプロイされます

3. **URLを確認**
   - 数秒でランダムなURLが生成されます
   - 例: `https://random-name-123.netlify.app`
   - このURLを会社の人に共有

### メリット
- ✅ GitHubアカウント不要
- ✅ 超簡単（ドラッグ&ドロップだけ）
- ✅ 無料
- ✅ すぐに公開される

### デメリット
- URLがランダム（カスタマイズ不可）

## 推奨方法

**会社のPCで見てもらうだけなら → GitHub Pages または Netlify Drop**

どちらも無料で、URLを共有するだけで見てもらえます。
特に Netlify Drop は GitHub アカウント不要で、最も簡単です。

## 注意事項

- 音声再生機能（Web Speech API）は、ブラウザによって動作が異なる場合があります
- Edge、Chrome、Safari で動作確認済み
- 会社のPCのブラウザが古い場合は、最新版に更新することを推奨
