# セキュリティの高い会社のPCでの使用方法

## ✅ このサイトの特徴

- **完全オフライン動作**: インターネット接続は不要です
- **外部サーバー不要**: すべてのファイルがローカルにあります
- **セキュリティポリシーに準拠**: 外部への通信は一切行いません

## 📦 必要なファイル（3つだけ）

以下の3つのファイルを同じフォルダに配置してください：

1. `index.html` - メインのHTMLファイル
2. `styles.css` - スタイルシート
3. `script.js` - JavaScriptファイル

## 🚀 使用方法

### 方法1: USBメモリで共有（推奨）

1. **USBメモリに3つのファイルをコピー**
   ```
   USBメモリ/
   ├── index.html
   ├── styles.css
   └── script.js
   ```

2. **各PCで実行**
   - USBメモリを挿入
   - `index.html` をダブルクリック
   - ブラウザが自動的に開きます

### 方法2: 共有フォルダで共有

1. **共有フォルダに3つのファイルをコピー**
   - 例: `\\server\shared\english-level-check\`

2. **各PCで実行**
   - エクスプローラーで共有フォルダを開く
   - `index.html` をダブルクリック

### 方法3: 各PCのデスクトップにコピー

1. **各PCのデスクトップに3つのファイルをコピー**
   - 例: `C:\Users\[ユーザー名]\Desktop\レベル判定\`

2. **実行**
   - `index.html` をダブルクリック

## ⚠️ 注意事項

### ブラウザについて

- **推奨ブラウザ**: Edge、Chrome、Firefox（最新版）
- **音声再生機能**: Web Speech APIを使用しています
  - Edge、Chrome: 正常に動作
  - Firefox: 音声再生が動作しない場合があります
  - Internet Explorer: 動作しません

### セキュリティについて

- このサイトは**外部への通信を行いません**
- データはブラウザ内（localStorage）にのみ保存されます
- サーバーへの送信は一切ありません

### トラブルシューティング

**Q: 音声が再生されない**
- EdgeまたはChromeを使用してください
- ブラウザを最新版に更新してください
- ブラウザの設定で音声合成が許可されているか確認してください

**Q: スタイルが適用されない**
- 3つのファイルが同じフォルダにあるか確認してください
- ファイル名が正確か確認してください（大文字小文字も含む）

**Q: エラーが表示される**
- ブラウザのコンソール（F12キー）でエラー内容を確認してください
- ブラウザを最新版に更新してください

## 📋 動作確認済み環境

- ✅ Windows 10/11 + Edge（最新版）
- ✅ Windows 10/11 + Chrome（最新版）
- ✅ オフライン環境
- ✅ ファイアウォール内のPC

## 🔒 セキュリティ情報

- **外部通信**: なし
- **データ送信**: なし
- **外部リソース読み込み**: なし
- **Cookie使用**: なし（localStorageのみ使用）
- **個人情報**: ブラウザ内のみに保存、外部送信なし

このサイトは完全にローカルで動作するため、セキュリティの高い環境でも安全に使用できます。
