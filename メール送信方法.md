# メールでのファイル送信方法

## 📊 ファイルサイズ確認

各ファイルのサイズは以下の通りです：

| ファイル名 | サイズ | 説明 |
|-----------|--------|------|
| `index.html` | **9.6KB** | メインのHTMLファイル |
| `styles.css` | **17KB** | スタイルシート |
| `script.js` | **45KB** | JavaScriptファイル |
| **合計** | **約72KB (0.07MB)** | 3ファイル合計 |

✅ **5MBの制限を大幅に下回っています！**

## 📧 メール送信方法

### 方法1: 個別に送信（推奨）

各ファイルを個別のメールで送信する方法です。

**手順：**
1. メール作成画面を開く
2. 件名: `英語レベル診断デモ - index.html`
3. 本文に説明を記載（例：下記参照）
4. `index.html` を添付
5. 送信

同様に `styles.css` と `script.js` も送信。

**本文例：**
```
お疲れ様です。

英語レベル診断のデモサイトのファイルをお送りします。

【ファイル1/3】index.html
このファイルをダブルクリックするとブラウザで開きます。
ただし、styles.css と script.js も同じフォルダに必要です。

次に styles.css と script.js も送信します。
```

### 方法2: ZIPファイルにまとめて送信（最も簡単）

3つのファイルをZIPにまとめて1回で送信する方法です。

**手順：**
1. 3つのファイルを選択（Ctrlキーを押しながらクリック）
   - `index.html`
   - `styles.css`
   - `script.js`
2. 右クリック → 「送信」→ 「圧縮（zip形式）フォルダー」
3. ZIPファイル名を変更（例: `英語レベル診断デモ.zip`）
4. メールに添付して送信

**本文例：**
```
お疲れ様です。

英語レベル診断のデモサイトのファイルをZIPでお送りします。

【使用方法】
1. ZIPファイルを解凍してください
2. 解凍したフォルダ内の index.html をダブルクリック
3. ブラウザで自動的に開きます

【注意事項】
- インターネット接続は不要です（完全オフライン動作）
- EdgeまたはChrome推奨
- 3つのファイルが同じフォルダにある必要があります

ご確認のほど、よろしくお願いいたします。
```

## 📦 ZIPファイルの作成方法（詳細）

### Windowsの場合

1. **エクスプローラーで3つのファイルを選択**
   - `index.html`
   - `styles.css`
   - `script.js`

2. **右クリック → 「送信」→ 「圧縮（zip形式）フォルダー」**

3. **ZIPファイル名を変更**
   - 例: `英語レベル診断デモ.zip`

4. **メールに添付**

### ZIPファイルのサイズ

- 3ファイル合計: 約72KB
- ZIP圧縮後: 約**20-30KB**程度（圧縮率による）

✅ **5MBの制限を大幅に下回っています！**

## ✅ メール送信のメリット

- ✅ ファイルサイズが小さい（合計0.07MB）
- ✅ 5MB制限内で余裕がある
- ✅ USBメモリ不要
- ✅ 複数の人に同時送信可能
- ✅ 送信履歴が残る

## ⚠️ 注意事項

### 受け取り側での注意

1. **3つのファイルを同じフォルダに配置**
   - ZIPで送った場合は、解凍すると自動的に同じフォルダになります
   - 個別に送った場合は、同じフォルダに保存してください

2. **ファイル名を変更しない**
   - `index.html`、`styles.css`、`script.js` の名前は変更しないでください

3. **ブラウザで開く**
   - `index.html` をダブルクリックするとブラウザで開きます
   - 推奨ブラウザ: Edge、Chrome

## 💡 推奨方法

**ZIPファイルにまとめて1回で送信する方法を推奨します。**

理由：
- 1回の送信で完了
- 3つのファイルが確実に同じフォルダに配置される
- 受け取り側が混乱しない
- ファイル名の変更ミスを防げる
