# 英会話教室 レベル診断機能付きホームページ 要件定義書（統合版）

## 📋 プロジェクト概要

### 目的
英会話教室のホームページに、**レベル診断機能を統合した来校促進型サイト**を構築する。ユーザーが自分の英語レベルを診断し、適切なコースを提案されることで、**無料体験レッスン・カウンセリングへの来校を促進**する。

### ターゲットユーザー
1. **英語初心者・中級者**（「喋れない」という悩みを持つ人）
2. **AIアプリで継続できなかった人**（モチベーションが上がらない人）
3. **具体的な目標を持つ人**（海外旅行、ビジネス、資格取得など）
4. **スマートフォンユーザー**（主にスマホでアクセス）

### 競合分析（ECC参考）
[ECC外語学院](https://www.ecc.jp/)の特徴：
- コースが豊富（初級～中級、中級～上級、ビジネス、TOEIC、英検など）
- 目的から探せる（趣味・留学、仕事で必要な英語力、資格取得など）
- 無料カウンセリング&無料体験レッスンを強調
- 受講生の声を掲載
- スクール検索機能

### 本サイトの差別化ポイント
1. **レベル診断機能**：来校前に自分のレベルを把握できる
2. **AIアプリで続かなかった人への対応**：対面レッスンの価値を提案
3. **初心者に優しい設計**：心理的安全性を確保
4. **来校促進に特化**：診断結果から無料体験への導線を明確に

### 4. レベル診断画面（既存機能を拡張）

#### 4.1 診断中の画面
- 進捗バー
- ステップ表示
- 問題表示
- 選択肢（ランダム化）
- 「わからない」ボタン
- 「間違えても大丈夫です」メッセージ

#### 4.2 音声再生機能（リスニング問題）
- 「🔊 音声を再生」ボタン
- 字幕は表示しない（リスニング問題として適切に）
- 音声再生失敗時のエラーハンドリング（「音声が再生できない場合は、スタッフにご相談ください」）
- 複数回再生可能

#### 4.3 スピーキング・ライティング問題の評価方法
**スピーキング問題：**
- 選択式問題（音声を聞いて適切な応答を選択）
- または、テキスト入力式（短い文章を入力）
- 評価は自動採点（正解/不正解）

**ライティング問題：**
- 選択式問題（適切な文章を選択）
- または、テキスト入力式（短い文章を入力）
- 評価は自動採点（正解/不正解）

**注意事項：**
- フェーズ1では、スピーキング・ライティングも選択式問題として実装
- フェーズ3以降で、音声認識・文章解析による高度な評価を検討

### 本要件定義の特徴
1. **ECCのような本格的な英会話教室サイト**を参考にした構成
2. **レベル診断機能を入口**として、来校促進に特化

1. **診断結果から無料体験への明確な導線**
2. **診断結果IDで来校時のスムーズな対応**
3. **おすすめコースの提案**（診断結果に基づく）
4. **無料体験レッスン・カウンセリングの強調**
